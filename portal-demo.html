üöÄ How to Use

For Students

  1. Click "Login" button in navigation
  2. Select "Student" tab
  3. Enter your Student ID and Password
  4. Click "Login"
  5. Access results portal and payment

Or create new account:
  1. Click "Sign Up" link
  2. Fill in your details
  3. Submit to create account

For Teachers/Admins

  1. Click "Login" button
  2. Select "Teacher/Admin" tab
  3. Enter your Staff ID and Password
  4. Click "Login"
  5. Access admin dashboard

Admin Features:
  • Manage students
  • Upload results
  • View payments
  • Generate reports

üìã Backend API Endpoints

Authentication

  • POST /api/auth/signup - Register new user
  • POST /api/auth/login - Login user
  • GET /api/auth/me - Get current user
  • POST /api/auth/forgot-password - Reset password

Payment

  • POST /api/payment/initialize - Start payment
  • GET /api/payment/verify/:ref - Verify payment
  • GET /api/payment/student/:id - Payment history

Results

  • POST /api/results/create - Create result
  • GET /api/results/student/:id - Get results
  • PUT /api/results/publish/:id - Publish result
